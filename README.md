# dip
Distributed Image Processing

# Setup

**Use vitualenv**
```
virtualenv venv
. ./venv/bin/activate
```

Setup the environment

```
sudo apt-get install fabric
fab setup_env

```
# Usage

### Process images in the input
```
fab process
```

### Clean the output
```
fab clean
```
